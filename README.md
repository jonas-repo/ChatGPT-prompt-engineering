# AI prompts for ChatGPT
This is a collection of prompts to use in ChatGPT for testing only.

## Basic Prompts

### Prompt Types
- **Prompts_specify_actions.txt**: Template for creating structured step-by-step plans, demonstrated through a software product launch plan example.

- **Prompts_structured_output.txt**: Examples of prompts that generate structured data in specific formats (JSON, Excel) for sentiment analysis.

- **Prompt_do_your_own_solution.txt**: Framework for evaluating student solutions by comparing against self-solved answers, demonstrated with an HTML table exercise.

- **Prompts_verify_conditions.txt**: Template for extracting and formatting instructions from natural language text.

- **Prompts_placeholders.txt**: System for creating personalized business emails using placeholder values.

- **Prompts_ICL.txt** (In-Context Learning): Examples for writing formal meeting invitation emails following specific templates.

- **Prompts_delimit.txt**: Guidelines for handling sensitive information and classifying incident urgency levels.

## Advanced Prompts

### Pattern Types
- **Prompts_root_prompt.txt**: Strict format rules for handling route queries between locations.

- **Prompt_person_pattern.txt**: Data protection expert simulation for filtering sensitive personal information.

- **Prompts_person_pattern2.txt**: Contains expert role-playing prompts for:
  - Cybersecurity analysis of potentially malicious emails
  - Vintage Linux terminal simulation

- **Prompts_pattern_mentaleng.txt**: Framework for converting natural language descriptions into structured pseudocode.

## Solutions Directory
- **chatGPT_Solutions/**: Contains example solutions generated by ChatGPT, including an HTML table implementation.
